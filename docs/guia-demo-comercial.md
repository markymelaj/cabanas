# Guía rápida para mostrar la demo

## Mensaje para enviar el link

Hola, soy Ignacio de Alto Cauce. Te dejo una demo simple de un sistema de reservas para cabañas.

La idea no es solo tener una web: el huésped consulta con fechas y datos completos, la solicitud queda guardada en un panel y el dueño puede confirmar, contactar por WhatsApp y controlar pagos/estados desde el celular.

Demo huésped: https://cabanas-theta.vercel.app/cabanas  
Panel demo: https://cabanas-theta.vercel.app/admin

Si te sirve, lo podemos adaptar a tus cabañas, fotos, precios, dominio y forma real de trabajar.

## Guion de venta en 60 segundos

1. Hoy muchas reservas se pierden porque llegan por WhatsApp incompletas.
2. Esta demo obliga a que la consulta llegue ordenada: fechas, huéspedes, cabaña y total estimado.
3. La solicitud queda guardada en el panel, no se pierde en el chat.
4. Desde el panel se confirma, se marca standby, se registra pago y se hace seguimiento.
5. El módulo de salón se ofrece solo si el cliente también vende eventos.

## Cierre sugerido

Lo importante es partir con algo útil y vendible: reservas de cabañas funcionando, panel simple y WhatsApp ordenado. Después se puede sumar pago online, salón, reportes o automatizaciones.
