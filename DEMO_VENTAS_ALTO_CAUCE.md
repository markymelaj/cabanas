# Presentación comercial — Alto Cauce Reservas

## Recorrido recomendado

1. Abrir `/` y explicar que el producto ordena consultas, disponibilidad, cobros y operación.
2. Entrar a `/cabanas` para mostrar la experiencia del huésped.
3. Elegir una unidad, dos fechas y revisar el total estimado.
4. Entrar a `/admin` para mostrar el panel responsive.
5. Revisar reservas, disponibilidad, pagos y eventos.

## Mensaje breve

Hola, soy Ignacio de Alto Cauce. Desarrollamos un sistema para alojamientos que reciben reservas por WhatsApp y necesitan ordenar disponibilidad, datos del huésped, pagos y seguimiento desde un panel simple.

La demo permite probar el recorrido completo como huésped y como administrador. Se entrega adaptada con las fotos, precios, reglas, dominio y WhatsApp de cada negocio.

## Cómo venderlo

No presentarlo como una página web. Presentarlo como un sistema operativo de reservas sin comisión, adaptado al negocio y usable desde el celular.
