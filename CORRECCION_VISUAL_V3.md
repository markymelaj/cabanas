# Corrección visual V3

Esta versión revierte el rediseño visual V2 y conserva la base funcional estable.

Cambios aplicados:

- Se eliminó la superposición de la franja de beneficios sobre el hero.
- Se eliminaron las tarjetas flotantes y decoraciones que competían con el contenido.
- Se recuperó la escala del panel de demostración anterior.
- El título queda controlado en dos líneas en escritorio.
- Se redujo la altura visual y el peso del primer pantallazo.
- Se recuperó la navegación simple anterior.
- Las secciones inferiores vuelven a la composición estable de la versión previa.
- Se mantiene el lockfile corregido para Vercel y el registry público en `.npmrc`.

Validaciones realizadas:

- ESLint aprobado.
- TypeScript aprobado.
- 4 pruebas funcionales aprobadas.
- Build de producción aprobado con Next.js 15.5.20.
